Leap Motion Controller Instruction
Author: Jiangyu Wang
Email: jiangyu@bu.edu
-----------------------------------------------------------------------------------------
Enviroment: Windows 64-bit System, Visual Studio

1. Make sure the all the included file in same folder
2. On Windows, open a Command Prompt and change to the above folder
3. Execute the following command: csc /platform:x86 /target:exe data.cs src\*.cs data.cs
4. Connecting the Leap Motion to Laptop
5. Run the following command in a command-line prompt: data.exe
